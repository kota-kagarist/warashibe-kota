# Contributing

改善提案を歓迎します。変更内容に応じて、次を使ってください。

- 交換提案: `交換提案` Issue Form
- サイトや文章の改善: `改善提案` Issue Form
- まだ結論を求めない相談: GitHub Discussions
- 具体的な修正: Issueを作成した後にPull Request

## 個人情報を投稿しない

公開リポジトリです。本名、住所、電話番号、メールアドレス、勤務先、追跡番号、受け渡し場所の詳細を投稿しないでください。誤って投稿した場合は、Issueの編集だけで済ませず、運営者へ削除を依頼してください。

## Pull Request

1. 関連Issueを記載する
2. 公開データだけを含める
3. `python3 scripts/check-public-data.py` を実行する
4. サイト表示をローカル確認する
