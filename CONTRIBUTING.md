# Contributing

改善提案と交換提案を歓迎します。公開プロジェクトですが、安全のため投稿できる情報を制限しています。

## 参加方法

- 交換提案: `交換提案` Issue Form
- サイトや文章の改善: `改善提案` Issue Form
- まだ結論を求めない相談: GitHub Discussions
- 具体的な修正: 関連Issueを作成した後にPull Request

交換提案の詳細条件は [`RULES.md`](RULES.md) を確認してください。

## 交換提案の流れ

1. 現在の募集が `Open` であることを確認する
2. Issue Formへ、モノの状態、付属品、不具合、交換理由を入力する
3. 個人情報を除いた写真を添付する
4. 公開質問へ回答する
5. 採用候補になった場合のみ非公開連絡へ移る
6. 双方確認後、匿名化した結果が公開記録へ残る

提案は先着順ではありません。採用されない場合や、情報不足・期限切れで閉じられる場合があります。

## 個人情報を投稿しない

公開リポジトリです。本名、住所、電話番号、メールアドレス、勤務先、追跡番号、受け渡し場所の詳細を投稿しないでください。写真の配送ラベル、顔、家の外観、車両番号、位置情報にも注意してください。

誤って投稿した場合は、Issueの編集だけで済ませず、運営者へ削除を依頼してください。Gitの履歴や通知に残る可能性があります。

## Pull Request

1. 関連Issueを記載する
2. 公開データだけを含める
3. ルール変更では理由と適用時期を記載する
4. `python3 scripts/check-public-data.py` を実行する
5. サイト表示をPCとスマートフォンで確認する
6. 交換記録の更新ではPrivacy Checkを実施する
