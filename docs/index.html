05 / RULES

続けるための約束。

面白さより安全を優先します。曖昧な状態のまま交換を進めません。

01

現金を条件にしない

モノとモノの交換が基本です。

02

状態を正直に共有する

傷、欠品、不具合を隠しません。

03

双方の納得を優先する

価格差だけでは判断しません。

04

危険なモノを扱わない

違法品、危険物、衛生上問題のある物は対象外です。

05

個人情報を公開しない

本名、住所、連絡先、生活圏を公開しません。

06

無理に交換しない

条件が合わなければ見送ります。

06 / OPEN PROJECT

企画そのものを、公開する。

交換提案、改善提案、判断記録、サイト変更はGitHubで管理します。交換募集が始まるまでは、改善提案を歓迎します。
